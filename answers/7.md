非负整数 x 二进制的所有位都相同，意味着 x=0 或者 x 的二进制全为 1，即 x=2^k - 1，其中 k 是非负整数。

问题变成求 [0,n] 中有多少个2^k - 1，或者说有多少个不同的 k。0≤2^k - 1≤n 即 1≤2^k≤n+1。

解得 0≤k≤w−1，其中 w 是 n+1 的二进制长度。

比如 n+1=10110，二进制长度为 5，那么 2^k≤10000，所以 k≤4。

所以有 w 个不同的 k。
```
class Solution {
public:
    int countMonobit(int n) {
        return bit_width((uint32_t) n + 1); //计算n+1的2进制位数(只支持无符号整型,故强制转换)
    }
};
```
