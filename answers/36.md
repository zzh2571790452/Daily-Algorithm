>&&的妙用

利用&&短路的效应充当if-else语句
```
class Solution {
public:
    int mechanicalAccumulator(int target) {
        target > 1 && (target += mechanicalAccumulator(target - 1));
        return target;
    }
};
```

>布尔数组

通过占用空间规避乘运算,通过右移1位规避除运算
```
class Solution {
public:
    int sumNums(int n) {
        bool arr[n][n+1];
        return sizeof(arr) >> 1;
    }
};
```
