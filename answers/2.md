>方法1:哈希表


```
class Solution {
public:
    int findRepeatDocument(vector<int>& documents) {
        unordered_map<int, bool> map;
        for(int doc : documents) {
            if(map[doc]) return doc;
            map[doc] = true;
        }
        return -1;
    }
};
```


>方法2:原地交换


题目说明,长度为n的数组中由0 ~ n-1 的数构成,所以索引与值为1对多.


所以通过遍历将索引与数字一一对应,当有第二个索引对应值等于索引时,该数重复
```
class Solution {
public:
    int findRepeatDocument(vector<int>& documents) {
        int i = 0;
        while(i < documents.size()) {
            if(documents[i] == i) {
                i++;
                continue;
            }
            if(documents[documents[i]] == documents[i])
                return documents[i];
            swap(documents[i],documents[documents[i]]);
        }
        return -1;
    }
};
```
优点:空间复杂度低
