>方法一: 字符串切片
```
class Solution {
public:
    string dynamicPassword(string password, int target) {
        return password.substr(target, password.size()) + password.substr(0, target);
    }
};
```


>方法二: 三次翻转
```
class Solution {
public:
    string dynamicPassword(string password, int target) {
        reverse(password.begin(),password.end());
        reverse(password.begin(),password.begin()+password.size()-target);
        reverse(password.end()-target,password.end());
        return password;
    }
};
```
